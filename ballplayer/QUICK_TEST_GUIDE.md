## 编译步骤

```
Quartus Prime → File → Open Project → ballplayer.qpf
```

### 2. 验证配置
```
确认顶层实体显示为：ballplayer_top_test
```

### 3. 执行编译
```
Processing → Start Compilation (或按 Ctrl+L)
```

### 4. 下载到FPGA
```
Tools → Programmer → Start
```

### LED状态：

## 🎯 测试判断

### ✅ 成功情况：
**看到四色条纹** → LCD硬件正常，问题在游戏代码
- 下一步：修复游戏图案数据
- 可以恢复正常游戏模式

### ❌ 失败情况：
**仍然白屏** → LCD初始化或连接问题
- 检查引脚连接
- 验证SPI信号

### ⚠️ 异常情况：
**显示其他图案** → 部分功能异常
- 记录具体现象
- 进一步诊断
